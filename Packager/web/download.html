Plugins

Marsatag Plug-in

MarsaTag-Plugin is a plugin to use the French POS-tagger MarsaTag directly from SPPAS. Notice that you have to install MarsaTag first.

By using the plugin, you'll get the following advantages:

  1. The POS-Tags are time-aligned; and
  2. it is ready-to-use (all steps to execute MarsaTag are pre-configured).

Download MarsaTag-PlugIn

TierMapping Plug-in

Create new tiers by mapping the labels of an existing tier, with new labels.

The following mapping tables are proposed:

  1. map the phonemes of the tier "PhonAlign" from SAMPA to IPA,
  2. map the phonemes of the tier "PhonAlign" to their properties (voiced/unvoiced, vowel/consonant, palatal/affricate etc).

Such tables are UTF-8 encoded in a CSV file, with columns separated by semicolon. They can be imported/modified/saved with OpenOffice, LibreOffice or Excel. The most important is to not change its properties (UTF8, semicolon) and location (in the plugin directory).

Download TierMapping-PlugIn

ESPPAS: deprecated

ESPPAS (in 2012) was supported by ANR OTIM project Ref Nr ANR-08-BLAN-0239. This was an add-on to SPPAS to deal with Enriched Orthographic Transcriptions, using LIA_Phon to perform grapheme-to-phoneme conversion.

Send an e-mail to the author to get a version -- not compatible with current version of SPPAS.
