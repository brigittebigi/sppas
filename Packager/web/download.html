Marsatag plugin

MarsaTag is a Part-Of-Speech tagging for French, e.g. Det, Noun, ... MarsaTag is developed at Laboratoire Parole et Langage, by S. Rauzy and G. De Moncheuil. Notice that you have first to install MarsaTag: download it here.

Using the plugin allows the following advantages:

  1. the resulting POS-Tags are time-aligned at tokens level; and
  2. it is ready-to-use, i.e. all steps to execute MarsaTag are pre-configured.

marsatagplugin-1.0

Sox plugin

SoX is the Swiss Army Knife of sound processing utilities. It can convert audio files to other popular audio file types and also apply sound effects and filters during the conversion. Notice that you have first to install sox: download it here.

By default, the plugin is configured to convert audio files into the format expected by SPPAS: 16 bits, 16000 Hz, 1 channel. However, such options can be modified and/or other options can be selected.

soxplugin-1.0

ESPPAS: deprecated

ESPPAS (in 2012) was supported by ANR OTIM project Ref Nr ANR-08-BLAN-0239. This was an add-on to SPPAS to deal with Enriched Orthographic Transcriptions, using LIA_Phon to perform grapheme-to-phoneme conversion.

Send an e-mail to the author to get a version -- not compatible with current version of SPPAS.
