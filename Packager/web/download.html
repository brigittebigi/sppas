SAMPA to IPA plugin

This plugin allows to convert transcriptions between the SAMPA (Speech Assessment Methods Phonetic Alphabet) annotation and the International Phonetic Alphabet.

It can convert either individual phonemes or strings of phonemes. The converted transcription is saved in a new file. Any symbols in the transcription tier which are not in the conversion file are not replaced.


SAMPA to Praat-IPA plugin

This plugin allows to convert transcriptions between the SAMPA (Speech Assessment Methods Phonetic Alphabet) annotation and the Praat codings of the International Phonetic Alphabet.

It can convert either individual phonemes or strings of phonemes. The converted transcription is saved in a new file. Any symbols in the transcription tier which are not in the conversion file are not replaced.


Classify phones plugin

This plugin allows to create annotations of the classification of phonemes: place of articulation, manner of articulation, phonation, etc. The classification is saved in a new file. Here is the list of tiers that will be created:

  1. Class: whether the phoneme is a vowel or a consonant
  2. Height: open, open-mid, close, etc. of vowels
  3. Backness: front, back, etc. of vowels
  4. Roundness: whether the vowel is rounded or unrounded
  5. Manner of articulation: in which way the sound is produced
  6. Place of articulation: where the major constriction happens inside the vocal tract
  7. Phonation: whether the phoneme is voiced or unvoiced
  8. Location: whether the consonant is oral or nasal
  9. Position: central, front, back, etc. of a consonant
  10. Airstream: whether the consonant is pulmonic or non-pulmonic


Marsatag plugin

MarsaTag is a Part-Of-Speech tagging for French, e.g. Det, Noun, ... MarsaTag is developed at Laboratoire Parole et Langage, by S. Rauzy and G. De Moncheuil. Notice that you have first to install MarsaTag: download it here.

Using the plugin allows the following advantages:

  1. the resulting POS-Tags are time-aligned at tokens level; and
  2. it is ready-to-use, i.e. all steps to execute MarsaTag are pre-configured.


Sox plugin

SoX is the Swiss Army Knife of sound processing utilities. It can convert audio files to other popular audio file types and also apply sound effects and filters during the conversion. Notice that you have first to install sox: download it here.

By default, the plugin is configured to convert audio files into the format expected by SPPAS: 16 bits, 16000 Hz, 1 channel. However, such options can be modified and/or other options can be selected.

ESPPAS (deprecated)

ESPPAS (in 2012) was supported by ANR OTIM project Ref Nr ANR-08-BLAN-0239. This was an add-on to SPPAS to deal with Enriched Orthographic Transcriptions, using LIA_Phon to perform grapheme-to-phoneme conversion.
