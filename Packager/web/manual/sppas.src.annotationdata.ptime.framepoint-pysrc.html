 55      """ 
 56      @author:  Brigitte Bigi

 58      @license: GPL, version 3 
 59      @summary: This class is the FramePoint representation. 
 60   
 61      Represents a frame point identified by a frame value and a radius value. 
 62      This is used to reference a placement in a media, depending on its 
 63      frame rate. 
 64   
 65      In the FramePoint instance, the 3 relations <, = and > take into 
 66      account a radius value, that represents the uncertainty of the placement. 
 67      For a point x, with a radius value of dx, and a point y with a radius 
 68      value of dy, these relations are defined as: 
 69          - x = y iff |x - y| <= dx + dy 
 70          - x < y iff not(x = y) and x < y 
 71          - x > y iff not(x = y) and x > y 
 72   
 73      Examples: 
 74   
 75          - Strictly equals: 
 76   
 77              - x = 3, dx=0. 
 78              - y = 3, dy=0. 
 79              - x = y is true 
 80   
 81              - x = 3, dx=0. 
 82              - y = 2, dy=0. 
 83              - x = y is false 
 84   
 85          - Using the radius: 
 86              - x = 3, dx=1. 
 87              - y = 2, dy=0. 
 88              - x = y is true (due to a vagueness of 1 frame around x) 
 89
