unpack(cls, other, noValue=None, noMargin=0.)
Class Method

source code 

Split a object into (value, margin)

Parameters:
  • other (a Delay, float, int, or many other things) - the object to split
  • noValue - (optional) value to return if any value is found. Default is None.
  • noMargin - (optional) margin to return if any margin is found. Default 0. nota: noMargin=None allow to know tets if a margin was found.
Returns:
a tuple of 2 'numerics' (value, margin) default value is None (=> other's value isn't a valid number) default margin is 0. (=> other hasn't a margin)
