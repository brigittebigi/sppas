 57      Represents a time point identified by a time value and a radius value. 
 58      Generally, time is represented in seconds, as a float value. 
 59   
 60      In the TimePoint instance, the 3 relations <, = and > take into 
 61      account a radius value, 
 62      that represents the uncertainty of the localization. 
 63      For a point x, with a radius value of dx, and a point y with a radius 
 64      value of dy, these relations are defined as: 
 65          - x = y iff |x - y| <= dx + dy 
 66          - x < y iff not(x = y) and x < y 
 67          - x > y iff not(x = y) and x > y 
 68   
 69      Examples: 
 70   
 71          - Strictly equals: 
 72   
 73              - x = 1.000, dx=0. 
 74              - y = 1.000, dy=0. 
 75              - x = y is true 
 76   
 77              - x = 1.00000000000, dx=0. 
 78              - y = 0.99999999675, dy=0. 
 79              - x = y is false 
 80   
 81          - Using the radius: 
 82   
 83              - x = 1.0000000000, dx=0.0005 
 84              - y = 1.0000987653, dx=0.0005 
 85              - x = y is true  (accept a margin of 1ms between x and y) 
 86
