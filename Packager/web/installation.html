Install SPPAS

The installation of SPPAS only consists to unpack the package.

In order to work, external programs must be installed before installing SPPAS. Follow carefully instructions of this page to download and install all of them: Python 2.7, wxPython, Julius. Notice that administrator rights are required to perform these installations.

Dependencies on Windows

  1. Click the link to download Python 2.7 "Windows x86 MSI installer", then execute it.
  2. Click the link to download wxPython with version name: "-win32-py27", then execute it.
  3. Click the link to download Julius (release >= 4.1). Unpack the file then go to into the directory. Open the "bin" folder then select the file "julius.exe". Copy this file into C:\WINDOWS\. Notice that the file name must be STRICTLY julius.exe, and nothing else (you'll have to rename it by removing the version number if any).

Dependencies on MacOS

  1. Click the link to download Python 2.7 "Mac OS X 64-bit/32-bit installer", then execute it.
  2. Click the link to download wxPython. Click on the package and follow instructions. Notice that this installer is not signed by Apple, so by default your Mac will tell you it is corrupted and will not install it. You can get around this by temporarily relaxing the security settings in System Preferences. In the Security & Privacy section set "Allow applications downloaded from" to "Anywhere."
  3. For Julius software, I prepared a 64-bits version which include a ready-to-use (old) version of Julius for MacOS. This version is available here: Julius for MacOS. Download, extract the archive and drag-and-drop the package content into the /usr/local/bin directory.

Dependencies on Linux

  • There are deb packages for all dependencies (python2.7, wxpython, julius). python-wxgtk-media and gstreamer must be installed too.
  • There are rpm packages for python2.7 and wxpython. The Julius tool will have to be installed from sources.
