SPPAS is a multi-platform free annotation tool that allows to create, visualize and analyze time-aligned annotations. It is able to produce automatically speech annotations from a recorded speech sound and its transcription. Some special features are also offered for the analysis of annotated files.

SPPAS is compatible with Praat, Elan, Transcriber, Annotation Pro, Phonedit, and many others...

Important information

Questions, problems, solutions:

  1. Make sure you have closely follow the installation guide and documentation.
  2. Many problems can be solved by updating your version.
  3. Take a look on the SPPAS User's group.
  4. If none of the above helps, you may send mail to the author. Indicate your operating system and versions, send the log file, and send a sample of the data on which a problem occurs.

Restrictions:

  • wav, aiff or au, mono, speech files only, 16 bits;
  • Annotation files: UTF-8 encoding only;
  • File names without special characters (no spaces, no accents, etc);
  • Exactly the same file name for the sound and for the transcription (except the extension).
